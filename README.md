# ![app icon](./.github/readme-images/app-icon.png) DyomiSY

DyomiSY is a fork of **Mihon**, based on TachiyomiSY. It is designed to provide
same experience as the original one while maintaining compatibility with the
upstream repository. This project was born as I needed to install two
TachiyomiSY application in the same device.

> [!IMPORTANT] This fork is in **maintenance mode**. I will not be adding new
> features, but I will keep the app updated with the latest changes and security
> patches from the original repository.

## Download

- **Stable:** Get the latest version from the
  [releases page](https://github.com/pazbryant7/tachiyomisy/releases/latest).
- **Preview:** Try upcoming updates
  [here](https://github.com/pazbryant7/tachiyomisypreview/releases).
